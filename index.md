# 개인정보처리방침 (Privacy Policy)

**확장 프로그램명:** 웨않되 - 맞춤법 검사기
**최종 수정일:** 2026-06-06

본 개인정보처리방침은 Chrome 확장 프로그램 "웨않되 - 맞춤법 검사기"(이하 "본 확장 프로그램")가 사용자 데이터를 어떻게 다루는지 설명합니다.

## 1. 수집·처리하는 데이터

본 확장 프로그램은 다음 데이터를 처리합니다.

### 1.1 맞춤법 검사 대상 텍스트
- 사용자가 맞춤법 검사를 위해 직접 선택하거나 입력한 텍스트(최대 800자)를 처리합니다.
- 이 텍스트는 맞춤법 검사 결과를 받기 위해 [외부 맞춤법 검사 서비스](https://alldic.daum.net/grammar_checker.do)로 전송됩니다.
- 본 확장 프로그램은 이 텍스트를 자체적으로 **저장하거나, 개발자의 서버로 전송하거나, 제3자에게 판매·공유하지 않습니다.** 검사 결과를 표시한 후 폐기됩니다.

### 1.2 로컬 설정 데이터
- 사용자가 "무시"하도록 설정한 맞춤법 항목 목록(`grammarBlacklist`, `grammarIgnores`)을 저장합니다.
- 이 데이터는 Chrome의 동기화 저장소(`chrome.storage.sync`)에만 저장되며, 개발자의 서버로 전송되지 않습니다.

### 1.3 쿠키
- 제휴 서비스 연동 여부를 확인하기 위해 특정 쿠키의 값을 읽습니다.
- 그 외의 쿠키는 읽지 않으며, 읽은 값을 외부로 전송하지 않습니다.

## 2. 데이터를 사용하지 않는 방식

본 확장 프로그램은 다음을 **하지 않습니다.**
- 사용자 데이터를 개발자 소유의 서버에 저장하지 않습니다.
- 사용자 데이터를 제3자에게 판매하거나 본래 목적과 무관하게 양도하지 않습니다.
- 신용도 평가나 대출 목적 등으로 사용하지 않습니다.
- 사용 통계(애널리틱스)나 추적을 수집하지 않습니다.

## 3. 제3자 서비스

맞춤법 검사 기능은 [외부 맞춤법 검사 서비스](https://alldic.daum.net/grammar_checker.do)를 이용합니다. 검사 대상 텍스트가 해당 서비스로 전송되며, [해당 서비스의 개인정보처리방침](https://www.kakao.com/policy/privacy)이 함께 적용됩니다.

## 4. 권한 사용 사유

- `storage`: 사용자의 맞춤법 무시 설정을 저장하기 위함
- `sidePanel`: 검사 결과를 사이드 패널 UI에 표시하기 위함
- `cookies`: 제휴 서비스 연동 여부 확인을 위함
- `host_permissions (<all_urls>)`: 사용자가 보고 있는 웹페이지에서 텍스트를 선택해 맞춤법 검사를 실행할 수 있도록 하기 위함

# Privacy Policy

**Extension name:** 웨않되 - Korean Spell Checker
**Last updated:** 2026-06-06

This Privacy Policy explains how the Chrome extension "웨않되 - Korean Spell Checker" (the "Extension") handles user data.

## 1. Data We Process

The Extension processes the following data.

### 1.1 Text to be spell-checked
- The Extension processes text (up to 800 characters) that the user explicitly selects or enters for spell checking.
- This text is sent to a [third-party spell-checking service](https://alldic.daum.net/grammar_checker.do) in order to obtain spell-check results.
- The Extension does **not store this text itself, does not transmit it to the developer's servers, and does not sell or share it with any third party.** It is discarded after the results are displayed.

### 1.2 Local settings data
- The Extension stores the list of spell-check items the user has chosen to "ignore" (`grammarBlacklist`, `grammarIgnores`).
- This data is stored only in Chrome's sync storage (`chrome.storage.sync`) and is not transmitted to the developer's servers.

### 1.3 Cookies
- The Extension reads the value of a specific cookie to check whether an affiliate service is connected.
- It does not read any other cookies, and does not transmit the value it reads externally.

## 2. What We Do Not Do

The Extension does **not**:
- Store user data on any developer-owned server.
- Sell user data to third parties or transfer it for purposes unrelated to its original purpose.
- Use data for purposes such as creditworthiness assessment or lending.
- Collect usage analytics or tracking.

## 3. Third-Party Services

The spell-check feature uses a [third-party spell-checking service](https://alldic.daum.net/grammar_checker.do). Because the text to be checked is sent to that service, [that service's privacy policy](https://www.kakao.com/policy/privacy) also applies.

## 4. Justification for Permissions

- `storage`: To save the user's spell-check ignore settings.
- `sidePanel`: To display spell-check results in the side panel UI.
- `cookies`: To check whether an affiliate service is connected.
- `host_permissions (<all_urls>)`: To allow the user to select text on the web page they are viewing and run a spell check.
